FDM comes with 2 different levels.

## Empty Level (L_FDM)

It is located in `Content/FlexibleDirectionalMenu/Levels`
![image90](/images/image90.png)

It just an empty level for testing different configurations quickly.

Select the character that is already placed on the level, open the `Details` panel and find `FDM|Testing` section, you will see some buttons.

![image91](/images/image91.png)
You can use these buttons to test the example configurations quickly.


## Demo Room (L_FDM_DemoRoom)

It is located in `Content/FlexibleDirectionalMenu/Demo/Levels`
![image92](/images/image92.png)

This map is designed to let you experience the features offered by the FDM system by playing it.

![image93](/images/image93.png)

Within the map, key features such as adding a new record, deleting an existing record, or moving its location during the work process are demonstrated practically.
